
## Local Setup

```sh
# Mac
brew install spatialite-tools
# M1 Mac
sudo ln -s /opt/homebrew/lib/mod_spatialite.dylib /usr/local/lib/mod_spatialite.dylib

# Linux
apt-get install libsqlite3-mod-spatialite
```
